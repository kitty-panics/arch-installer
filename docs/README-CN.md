Arch Linux 快速安装脚本。

## 使用

如果你想要快速的一键安装 Arch Linux 系统，执行下面的命令即可。

```Bash
bash <(curl -sL https://raw.githubusercontent.com/kitty-panics/arch-installer/master/bin/setup.sh)
```

**备注：**

执行这条命令前，请确认你的设备满足以下条件。

+ 设备处于一个 ``高速且稳定`` 的网络环境中。
+ 设备中的 ``重要文件`` 已备份。
+ 仔细阅读安装脚本中的提示信息。
